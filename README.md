# Text Search Assistant - 文本搜索助手

这是一个 Chrome 扩展程序，当您在网页上选中文本时，会在文本旁边显示搜索图标，点击图标可以直接跳转到相应平台进行搜索。

## 功能特点

1. 在任何网页上选中文本时自动显示搜索图标
2. 支持多个平台：京东、淘宝、闲鱼、Google
3. 图标会根据空间自动横向或纵向排列
4. 可以在选项页面配置启用的平台
5. 可以设置忽略网址，在特定网站不显示搜索图标
6. 图标加载失败时自动降级为emoji显示
7. 图标尺寸适中，不会遮挡页面内容
8. 智能识别文本，避免在不合适的场景下显示（如网址、邮箱等）

## 智能识别功能

为了避免不必要的干扰，扩展会在以下情况不显示搜索图标：
- 选中的文本包含网址链接（不仅限于完整网址，也包括部分包含网址的文本）
- 选中的文本是邮箱地址
- 选中的文本是纯数字
- 选中的文本包含过多特殊符号
- 选中的文本为空或只有一个字符

## 安装方法

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击左上角的「加载已解压的扩展程序」
4. 选择本项目文件夹
5. 安装完成

## 使用方法

1. 在任何网页上用鼠标选中文本
2. 在文本附近会出现搜索图标
3. 点击想要搜索的平台图标
4. 自动在新标签页中打开该平台的搜索结果页面

## 配置选项

点击扩展图标，选择「选项」进入设置页面：

1. 启用/禁用特定平台的搜索图标
   - 每个平台都有对应图标显示
   - 内容居中对齐，美观直观
2. 添加不想显示搜索图标的网址（例如已经在购物网站中就不需要显示）

## 替换图标

目前项目中的图标都是占位符文件，如需替换请按照以下对应关系替换 `/icons` 目录下的文件：

### 扩展图标（必须替换以获得良好外观）
- `icon16.png` - 16x16 扩展工具栏图标
- `icon48.png` - 48x48 扩展管理页面图标
- `icon128.png` - 128x128 Chrome网上应用店图标

### 平台图标（可选替换）
- `jd.png` - 京东平台图标
- `taobao.png` - 淘宝平台图标
- `xianyu.png` - 闲鱼平台图标
- `google.png` - Google平台图标

建议使用透明背景的 PNG 格式图标，尺寸为 20x20 像素以获得最佳效果。

如果图标文件缺失或无法加载，扩展会自动降级为显示emoji图标：
- 京东: 🐶
- 淘宝: 🎁
- 闲鱼: 🐠
- Google: 🔍

## 文件说明

- `manifest.json` - 扩展配置文件
- `content.js` - 内容脚本，处理文本选择和图标显示
- `content.css` - 内容样式，控制图标外观
- `options.html/js` - 选项页面，用于配置扩展
- `popup.html/js` - 弹出页面，提供基本说明和设置入口
- `background.js` - 后台脚本，处理扩展初始化
- `/icons` - 图标文件目录

## 技术细节

- 使用 Chrome Extensions Manifest V3
- 使用 Storage API 保存用户设置
- 使用 Content Scripts 实现页面交互
- 使用 CSS Flexbox 实现图标自适应排列
- 提供图标加载失败的降级方案
- 智能文本识别避免无效显示
- 支持包含网址的文本识别
